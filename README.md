# Lesson Learning Public Site

Static public website for `lessonlearning.com`.

This repository is intentionally small because it is meant for GitHub Pages only.

## Contains

- Root launchpad at `/`
- Teacher Lara Pires page at `/teacher-lara-pires/`
- Public images and CSS used by those pages
- `CNAME` for the custom domain

## Does Not Contain

- Lesson Learning portal app code
- Prisma schema or database files
- Docker/Lightsail deployment files
- Secrets or production environment files
- Private project documentation

## Publishing

GitHub Pages can serve this repository from the `main` branch root.
